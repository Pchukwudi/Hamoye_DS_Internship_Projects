# Hamoye_DS_Internship_Projects

The dataset is provided by the Food and Agriculture Organization of the United 
Nations and is made available in .csv format on the github link below:
https://github.com/HamoyeHQ/HDSC-Introduction-to-Python-for-machine-learning

This project is centered on data manipulation to demonstrate mastery of 
the core concept presented in the course module: Introduction to Python for Machine Learning
The concepts include: Numpy Array and vectorization, Pandas, Data Types and Data Wrangling
